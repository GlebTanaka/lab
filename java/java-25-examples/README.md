Java 25 — Examples Index

This repository hosts small, focused examples of Java 25 features and modern patterns.

Examples:

1) 00_maint_method — Entry point methods
- Shows the new unnamed‑class `void main()` and the classic `public static void main(String[] args)`.
- How to run from project root:
  - New style: `java 00_maint_method\src\Main.java`
  - Classic style: `java 00_maint_method\src\OldStyleMain.java someArg`
- Details: see `00_maint_method/README.md`.

2) 01_flexible_constructors — Flexible constructors
- Demonstrates modern constructor flexibility, including statements before `super(...)`.
- How to run from project root:
  - `java 01_flexible_constructors\src\FlexibleConstructorsDemo.java`
- Details: see `01_flexible_constructors/README.md`.

Notes:
- Windows PowerShell paths are used above. Adjust for your shell/OS if needed.
- Some legacy files may exist under `src/` from initial setup; prefer running examples from their numbered folders.
